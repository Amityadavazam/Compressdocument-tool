Aadhaar PDF Password Format (For Authorized Access)

Example 2: Name = "Suresh", Birth year = "1990" → Password = SURE1990

Example 3: Name = "Priyanka", Birth year = "1978" → Password = PRIY1978

Authorized Access Tip: These variations may work with proper authorization:
