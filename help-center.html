Advanced FAQs

Detailed answers to technical questions about our tools

Local processing means all file operations happen directly in your web browser using JavaScript. Your files never leave your computer or get uploaded to any server. This ensures complete privacy since there's no way for anyone else to access your documents. The trade-off is that processing very large files might be slower depending on your device's capabilities.

For free users: PDF files up to 50MB, images up to 20MB. For Premium users: PDF files up to 500MB, images up to 100MB. These limits are based on browser memory constraints. If you need to process larger files, consider splitting them first or upgrading to Premium for batch processing capabilities.

Our unlock tool works on PDFs with standard password protection (user password). However, PDFs with owner passwords (for editing restrictions) or encrypted with advanced security certificates (like those from banks) may not be unlockable. For such files, you need the original password. We don't support brute-force attacks as they're legally questionable.

Batch processing is available for Premium users. With a Premium subscription, you can process multiple files simultaneously in tools like Merge PDF, Image to PDF, Compress Image, and Convert Format. Free users need to process files one at a time. Batch processing significantly improves productivity when working with multiple documents.

First, try refreshing the page and clearing your browser cache. Make sure you're using an updated browser (Chrome, Firefox, Edge, or Safari). If the problem persists, check if JavaScript is enabled in your browser settings. For file-specific issues, try a different file to see if the problem is with that particular file. If nothing works, contact our support team with details about the issue.
