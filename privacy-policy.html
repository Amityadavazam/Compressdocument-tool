1. Introduction

Welcome to Compress Document Pro Suite's Privacy Policy. This document explains how we handle your information when you use our free online tools.

Core Principle: We believe in "Privacy by Design". Our tools are built to protect your privacy from the ground up.

1.1. Scope

This policy applies to all users of Compress Document Pro Suite website and services, including:

  • PDF processing tools
  • Image editing tools
  • Document conversion tools
  • All other tools available on our platform

1.2. Definitions

  • "Service": Compress Document Pro Suite website and tools
  • "User": Anyone accessing or using our Service
  • "Personal Data": Information that identifies an individual
  • "Processing": Any operation performed on Personal Data

2. Information We Collect

We collect minimal information necessary to provide and improve our services:

2.1. Information You Provide

Data Type Purpose Storage
Email Address (if contacted) Support communication Until request resolved
Feedback/Comments Service improvement Anonymous storage

2.2. Automatically Collected Information

  • Usage Data: Tool usage statistics (anonymized)
  • Technical Data: Browser type, version, device type
  • Performance Data: Load times, error rates
  • Location Data: Country-level (for statistics only)

Important: We DO NOT collect or store your uploaded files. All file processing happens locally in your browser.

2.3. What We Never Collect

  • Your uploaded files (PDFs, images, documents)
  • Personal identification from files
  • Passwords from protected documents
  • Credit card or payment information
  • Social media accounts

We use collected information only for specific, legitimate purposes:

3.1. Service Provision

  • Comply with legal obligations

  • Ensure service security

4. Data Protection

We implement robust security measures to protect your information:

4.1. Security Measures

  • HTTPS Encryption: All data transmitted over SSL/TLS
  • Browser Security: Leveraging modern browser security features
  • Regular Audits: Security reviews of our codebase
  • Access Control: Limited access to collected data

4.2. Data Retention

4.3. International Data Transfers

As a cloud-based service, data may be processed in various locations. We ensure:

  • All processors comply with data protection standards
  • Adequate safeguards for international transfers
  • GDPR compliance for EU users
  • CCPA compliance for California users

5. File Processing Security

This is our most important privacy feature - local file processing:

Key Feature: All file processing happens 100% in your browser. Files never leave your device.

5.1. How It Works

  • Files are loaded directly into your browser's memory
  • Processing occurs using JavaScript libraries
  • Results are generated locally
  • Files are cleared from memory when you leave the page

5.2. Technical Implementation

  • PDF Processing: Using pdf-lib and pdf.js libraries
  • Image Processing: Canvas API and Web Workers
  • Document Conversion: Client-side libraries
  • Security: Sandboxed browser environment

5.3. Benefits

  • Privacy: No server access to your files
  • Speed: Faster processing without uploads
  • Security: No risk of server breaches
  • Control: You maintain full control of your data

Important Note: While we implement maximum security, ensure you trust the files you upload. Malicious files could affect your local system.
